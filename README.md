An Arduino-based automatic vacuum cleaner robot that avoids obstacles using an ultrasonic sensor.

## Features
- Automatic movement
- Obstacle avoidance
- Continuous vacuum cleaning
- Low-cost hardware

## Components Used
- Arduino UNO
- Ultrasonic Sensor (HC-SR04)
- L298N Motor Driver
- DC Motors
- Vacuum Motor

## Working Principle
The robot moves forward while checking obstacles. If an obstacle is detected, it turns automatically and continues cleaning.
